# Automated Multi-Worm Counting Using OpenCV

This project implements an OpenCV-based pipeline for automated counting and tracking of *C. elegans* worms in video frames.

## 🔬 Overview
Manual worm counting is time-consuming and error-prone. This pipeline automates detection and tracking using:
- OpenCV for image processing
- Kalman Filter for position prediction
- Hungarian Assignment Algorithm for ID consistency
- Trajectory smoothing with Savitzky-Golay filter

## ⚙️ Features
- Multi-worm detection and ID assignment
- Per-frame statistics (CSV)
- Annotated tracking video
- Statistical plots and visualizations

## 📂 Files
- `multi_worm_counting.ipynb` — main analysis notebook  
- `worm_counts_per_frame.csv` — per-frame worm counts  
- `worm_tracking_output.mp4` — annotated tracking video  
- `worm_count_stats.jpeg` — visual summary
